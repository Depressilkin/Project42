команды для командной строки:
cd - перейти в директорию
mkdir - создать директорию 
git config --global user.name - задать имя
git config --global user.email - задать почту
git init - инициализировать репозиторий
git status - проверка статуса репозитория
git add (имя файла или .) - добавление (подготовка к записи) изменений
git commit -m "описание изменения"- запись изменения
git log - просмотр истории коммитов
git checkout (адрес поинта) - переход на поинт
git master - возврат в мастер ветку
Это изменения из новой ветки!
